\textbf{Задача №1}
Имеется не более чем счётный набор положительных действительных чисел $0 < \alpha_i < 1$ таких, что $\displaystyle\sum_{i} \alpha_i = 1$, заданный диаграммой переходов автомата. Решить задачу аналитически: построить попарно непересекающиеся открытые множества $U_i$ в $\mathbb{Z}_2$ такие, что $\mu(U_i) = \alpha_i$ для всех $i$, где $\mu$ --- нормализованная мера Хаара.
\begin{figure}[!ht]
	\centering
	\includegraphics[width=10cm]{Диаграмма.png}
	\caption{Диаграмма переходов автомата}
\end{figure}

\textbf{Аналитическое решение.}
Для представленного на рисунке автомата можно выделить одну эргодическую компоненту $\mathfrak{G}_1 = \{s_3, s_4\}$, которая достигается из бесконеных множеств $C_1 = \{000, 0010, 00110,\dots, 0011\dots10\}$,\\ $C_2 = \{010, 0110, 01110,\dots, 0111\dots10\}$, $C_3 = \{100, 1010, 10110,\dots, 1011\dots10\}$. Из конечного множества $C_4 = \{11\}$.

Для $C_1$ вероятность перехода в $\mathfrak{G}_1$ равна $\frac{1}{8} + \frac{1}{16} + \frac{1}{32} + \dots = \frac{1}{4}$.

Для $C_2$ вероятность перехода в $\mathfrak{G}_1$ равна $\frac{1}{8} + \frac{1}{16} + \frac{1}{32} + \dots = \frac{1}{4}$.

Для $C_3$ вероятность перехода в $\mathfrak{G}_1$ равна $\frac{1}{8} + \frac{1}{16} + \frac{1}{32} + \dots = \frac{1}{4}$.

Для $C_4$ вероятность перехода в $\mathfrak{G}_1$ равна $\frac{1}{4}$.

Исходя из этого, множество $C = C_1 \cup C_2 \cup C_3 \cup C_4$ является максимальным префиксным кодом ($\sum_{w \in C} = 1$), следовательно, все бесконечные слова с префиксом из $C$ будут задавать разиение $\mathbb{Z}_2$ на счётное число попарно-непересекающихся шаров ненулувого радиуса:
\[
    \mathbb{Z}_2 = [B_{\frac{1}{8}}^{000} \cup B_{\frac{1}{16}}^{0100} \cup B_{\frac{1}{32}}^{01100} \cup \dots] \cup [B_{\frac{1}{8}}^{010} \cup B_{\frac{1}{16}}^{0110} \cup B_{\frac{1}{32}}^{01110} \cup \dots] \cup [B_{\frac{1}{8}}^{001} \cup B_{\frac{1}{16}}^{0101} \cup B_{\frac{1}{32}}^{01101} \cup \dots] \cup [B_{\frac{1}{4}}^{11}]
\]
где шар $B_{\frac{1}{8}}^{000}$ содержит все 2-адические целые числа с префиксом 000, шар $B_{\frac{1}{8}}^{0100}$ содержит все 2-адические целые числа с префиксом 0100 и т.д.

В пространстве $\mathbb{Z}_2$ нормализованная мера Хаара для шара радиуса $p^{-k}$ с центром в точке $a \in \mathbb{Z}_2$ равна его радиусу, т.~е. $\mu_2(B_{2^{-k}}) = 2^{-k}$. Тогда множество $C$  также задаёт счётный набор действительных чисел $A = \{\mu_2(B_{\alpha_1}), \mu_2(B_{\alpha_2}), \mu_2(B_{\alpha_3}), \dots\} = \{\alpha_1, \alpha_2, \alpha_3, \dots\} = \{\frac{1}{4}, \frac{1}{8}, \frac{1}{16}, \frac{1}{32}, \dots, \frac{1}{8}, \frac{1}{16}, \frac{1}{32}, \dots, \frac{1}{8}, \frac{1}{16}, \frac{1}{32}\dots\}$, где $\sum_{a \in A} = 1$ в силу нормализованности меры Хаара $\mu_2(\mathbb{Z}_2) = 1$.

При бесконечной подаче единицы в состоянии $s_5$ автомат никогда не перейдёт в эргодическую компоненту. Вероятностная мера слова бесконечной длина равна нулю, следовательно, оно задаст шар нулевого радиуса.
